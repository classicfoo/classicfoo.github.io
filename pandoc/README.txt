Using Pandoc To Convert Text Files to HTML 
Author: Michael Huynh
Created: 16/04/22
Modified: 16/04/24

Steps:
1. Format your text file with Markdown
2. Place your markdown file into the folder "D:\para\areas\blog\pandoc"
3. Run the file "markdown_to_html.bat"
4. Move output file into your blog directory
